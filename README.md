# Go.USA.govThis is a repo for managing machine readable artifacts for the [Go.USA.gov APIs](https://go.usa.gov/api). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Go.USA.gov APIs](https://go.usa.gov/api).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://go.usa.gov/api)- [Developer](https://go.usa.gov/api)- [Twitter](https://twitter.com/USAgov)The [APIs.json](https://github.com/api-evangelist/go-usa-gov/blob/master/apis.json) index provides links to all the relevant links for the Go.USA.gov APIs, including the OpenAPI and Postman if they are present.
